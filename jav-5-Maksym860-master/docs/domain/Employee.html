Клас Employee

java.lang.Object
   └─ domain.Employee

public class Employee extends java.lang.Object

Представляє загального працівника компанії. Це базовий клас для всіх типів працівників. Кожен працівник має унікальний випадково згенерований ідентифікатор, ім'я, посаду, рівень кваліфікації (1–3) та відділ.

@author Maksym    @version 1.0

Деталі конструкторів

Employee()
Створює працівника з випадково згенерованим ідентифікатором (0–999). Інші поля залишаються незаповненими.

Employee(String name, String jobTitle, int level, String dept)
@param name повне ім'я (якщо формат невірний — встановлюється "John Doe")
@param jobTitle посада працівника
@param level рівень кваліфікації 1–3; інші значення → 1
@param dept назва відділу

Деталі методів

setName(String name)
Встановлює ім'я працівника. Ім'я повинно містити ім'я та прізвище через пробіл, лише літери, апострофи, коми, крапки або дефіси. Якщо валідація не пройшла — встановлюється "John Doe".
@param name повне ім'я працівника

setLevel(int level)
Встановлює рівень кваліфікації. Приймає 1, 2 або 3; будь-яке інше значення тихо замінюється на 1.
@param level рівень кваліфікації
