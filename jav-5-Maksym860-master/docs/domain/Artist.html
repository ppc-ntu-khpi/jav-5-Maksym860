Клас Artist

java.lang.Object
   └─ domain.Employee
       └─ domain.Artist

public class Artist extends Employee

Представляє працівника-художника, який має набір творчих навичок. Artist розширює Employee і додає список навичок, що описують художні здібності працівника.

@author Maksym    @version 1.0    @see Employee

Конструктори

КонструкторОпис
Artist()Створює художника з випадковим ідентифікатором та порожнім масивом навичок (місткість 10).
Artist(String[] skiils)Створює художника із зазначеними навичками та випадковим ідентифікатором.
Artist(String[] skiils, String name, String jobTitle, int level, String dept)Створює художника з усіма зазначеними даними.
