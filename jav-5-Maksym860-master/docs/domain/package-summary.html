API Працівників Компанії

Огляд Пакет Клас

Пакет domain

Містить класи, що моделюють структуру невеличкої компанії: працівників, менеджерів, художників та спеціалізовані ролі.

Перелік класів

КласОпис
EmployeeБазовий клас загального працівника компанії.
ArtistПрацівник з набором творчих навичок; розширює Employee.
EditorХудожник з перевагою електронного або паперового редагування; розширює Artist.
GraphicIllustratorСпеціаліст з графічної ілюстрації; розширює Artist.
TechnicalWriterСпеціаліст з технічної документації; розширює Artist.
ManagerМенеджер, що керує командою; розширює Employee.
