Клас Manager

java.lang.Object
   └─ domain.Employee
       └─ domain.Manager

public class Manager extends Employee

Представляє менеджера, який керує командою об'єктів Employee. Manager розширює Employee і зберігає масив підпорядкованих працівників.

@author Maksym    @version 1.0    @see Employee

Конструктори

КонструкторОпис
Manager()Створює менеджера з випадковим ідентифікатором та порожньою командою (місткість 10).
Manager(Employee[] employees)Створює менеджера з командою; інші поля незаповнені.
Manager(Employee[] employees, String name, String jobTitle, int level, String dept)Повний конструктор з усіма даними та командою.
